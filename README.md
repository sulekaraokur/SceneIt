SceneIt

SceneIt is an Android app that lets you look up TV series, save the ones you like, and keep track of your watching progress in a simple and clean way. The goal of the project was to build something functional, organized, and easy to navigate while using modern Android tools and design principles.

Main Features

- Search TV series through the TMDB API
- Save shows to a local library with Room
- Track watched episodes and overall progress
- Mark favorite shows with a separate layout
- Add your own notes and ratings
- Custom views for rating and episode progress

- Double-tap and swipe for gestures
